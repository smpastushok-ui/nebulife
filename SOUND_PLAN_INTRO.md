# План звуку — перша сесія (інтро + онбординг)

Карта всіх звуків на шляху нового гравця: від завантаження до початку туторіалу.
Розділ «ЗГЕНЕРУВАТИ» — список для генерації. Технічні вимоги — внизу.

## Шлях гравця

```
1. StarBirthIntro (генезис: сингулярність → інфляція → зоря → кластер, WebGL)
2. Екран входу (Apple / Google / Гість)
3. CinematicIntro:
   3.1 Субтитри над всесвітом + кнопка «ПОЧАТИ ГРУ»
   3.2 Варп-перехід (затемнення 2.2 с)
   3.3 Слайд 0 — аварійна передача (червоний термінал, NEW текст замість відео)
   3.4 Слайд 1 — термінал ЦУП (дані системи + загроза)
   3.5 Слайд 2 — директива евакуації (зелений термінал, NEW текст замість відео)
   3.6 Слайд 3 — «Доля народу у ваших руках» + кнопка «Почати місію»
4. Галактика → туторіал (там звуки вже є)
```

## ЗГЕНЕРУВАТИ (нові звуки)

Файли класти в `packages/client/public/sfx/` у двох форматах: `.webm` (Opus) + `.mp3`.

### 1. `intro-genesis-loop` — «Гул народження Всесвіту»
- **Де**: StarBirthIntro — фон усієї сцени генезису (~10 с, але робити лупом).
- **Що**: глибокий низькочастотний космічний дрон, повільно «дихає», без мелодії.
  Відчуття порожнечі, з якої щось народжується.
- **Формат**: луп 30–60 с, безшовний. Тихий (буде грати на 20–30% гучності).

### 2. `intro-bigbang` — «Спалах сингулярності»
- **Де**: StarBirthIntro — момент, коли спалахує зоря (кульмінація генезису).
- **Що**: суббасовий удар + широкий whoosh, що розгортається в дзвінкий шлейф.
  Як «великий вибух», але елегантно, не лякаюче.
- **Формат**: one-shot 2–3 с.

### 3. `intro-star-chime` — «Народження зорі»
- **Де**: StarBirthIntro — поява навігаційних ниток / кластера наприкінці.
- **Що**: кришталевий м'який дзвін / арпеджіо 2–3 ноти, «магічна точка світла».
- **Формат**: one-shot 1–1.5 с.

### 4. `intro-theme` — «Тема входу» (опційно)
- **Де**: StarBirthIntro + екран входу (Apple/Google/Гість). Зараз там ПОВНА тиша
  (стару мелодію інтро вимкнули — якщо хочеш музику тут, потрібна нова, спокійніша).
- **Що**: атмосферна ембієнт-тема, мінімалістична, без ударних. Налаштовує на
  «космічну подорож», не відволікає від кнопок входу.
- **Формат**: луп 60–90 с, безшовний.

### 5. `directive-stamp` — «Директиву прийнято»
- **Де**: CinematicIntro, слайд 2 — поява панелі «ДИРЕКТИВА ЕВАКУАЦІЇ»
  (новий текстовий слайд замість відео брифінгу).
- **Що**: короткий низький «штамп» / удар печатки + металевий відгук.
  Відчуття офіційного наказу.
- **Формат**: one-shot 0.7–1 с.

### 6. `mission-start` — «Початок місії»
- **Де**: CinematicIntro, слайд 3 — натискання кнопки «Почати місію»
  (найважливіша кнопка онбордингу, перехід у гру).
- **Що**: героїчний короткий ризер + світлий акорд, «двигуни запущено».
  Емоційний підйом, нагорода за проходження інтро.
- **Формат**: one-shot 2–3 с.

## Технічні вимоги

- **Формати**: `.webm` (Opus, ~96 kbps) + `.mp3` (fallback для Safari/iOS) — обидва файли з однаковою назвою.
- **Папка**: `packages/client/public/sfx/` (музика-лупи можна в `/music/`).
- **Лупи**: безшовні (без клацання на стику), перевірити на повторі.
- **Гучність**: нормалізувати приблизно до -16 LUFS; гучність у грі регулюємо кодом.
- **Обмеження браузера**: звук реально стартує лише після першого дотику/кліку
  гравця (autoplay policy) — для StarBirthIntro це означає, що дрон підхопиться
  з моменту першої взаємодії; це нормально.
