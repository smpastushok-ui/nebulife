## Фактична відповідь: Скільки типів космічних подій?

### Результат перевірки `cosmic-catalog.ts` + папки `public/cosmic-events/common/`

**Розробник помилявся — каталог дуже великий**, але реально "видними" новому гравцю є лише ті, що вже мають готові зображення.

| Метрика | Значення |
|---|---|
| **Усього записів у каталозі** | **136** |
| **Rollable (не eventOnly)** | **116** (нормальні дослідження) |
| **eventOnly** | **21** (1 `comet-herald` + 20 сезонних аномалій) |
| **З готовим pre-gen артом** | **19 зображень** у `public/cosmic-events/common/` |
| **Категорій** | **11** (туманності, зірки, галактики, феномени, екзопланети, темні об'єкти, зіркоутворення, подвійні зірки, малі тіла, бродячі об'єкти, сезонні) |

**Розподіл за рідкістю (116 rollable):**
- Common: 19 | Uncommon: 28 | Rare: 46 | Epic: 24 | Legendary: 20

**19 об'єктів із готовим артом (`common/`):**
`white-dwarf`, `brown-dwarf`, `red-dwarf`, `flare-star`, `lenticular-galaxy`, `irregular-galaxy`, `dwarf-galaxy`, `stellar-flare`, `bow-shock`, `super-earth`, `sub-neptune`, `molecular-cloud`, `open-cluster`, `eclipsing-binary`, `spectroscopic-binary`, `comet`, `asteroid-belt`, `l-type-brown-dwarf`, `t-type-brown-dwarf`

**Висновок:** каталог об'єктивно великий (116 типів), але **без pre-gen арту** для 97 типів — гравці бачать їх лише якщо витратять кварки або переглянуть рекламу (Tier-1 генерація). Для нових гравців ефективно доступні лише 19 зображень безкоштовно. Проблема не в кількості типів, а в **економіці доступу**.

## Рекомендація: Нові типи подій або послаблення кварок-економіки?

### Аналіз кварк-економіки (поточний стан)

**Джерела кварків:**
| Джерело | К-сть |
|---|---|
| Реєстрація (стартові) | 50⚛ (сервер) / показується 30⚛ (клієнт-тост — баг!) |
| Щоденний логін | 1⚛/день |
| Щоденні директиви | 1⚛ (3⚛ на 7-й день стріку) |
| Rewarded реклама | **1⚛/оголошення**, макс 10/день |
| Академія — квест | 1⚛ |
| Академія — вікторина | 2⚛ |
| Precursor-карта (дублікат) | 2⚛ |
| Precursor — повна колекція | 100⚛ (одноразово) |
| Season complete | 150⚛ (кожні 7 тижнів) |
| Мегаструктура (пул) | ~5–2000⚛ пропорційно |
| Premium ("Pro") | 5⚛/день |

**Витрати кварків (зберігач):**
| Призначення | Вартість |
|---|---|
| Фото uncommom | 30⚛ |
| Фото rare | 45⚛ |
| Фото epic | 60⚛ |
| Фото legendary | 80⚛ |
| Видобуток без реклами | ~10⚛ (орієнтовно) |

**Проблема:** безкоштовний гравець заробляє ~2–3⚛/день (логін + директива + можливо академія). Щоб побачити epic-фото без реклами — потрібно 20–30 днів. При цьому каталог має 24 epic-об'єкти і 20 legendary. Сезонна нагорода (150⚛) — найбільша безкоштовна виплата, але вона раз на 7 тижнів.

**Конкретні пропозиції (без змін пейволу):**

1. **Виправити стартовий тост:** показувати 50⚛ замість 30⚛ (баг у `App.tsx`, `STARTER_QUARKS_CLIENT = 30`).
2. **Перший discovery безкоштовно:** якщо у гравця ще немає жодного фото, перша AI-генерація незалежно від рідкості — безкоштовна (вау-момент онбордингу).
3. **Підняти щоденний логін** з 1⚛ до 2⚛ (подвоєння пасивного потоку без зламу балансу).
4. **Стрік-бонус директив:** підняти з 3⚛ до 5⚛ на 7-й день.
5. **Milestone-бонуси:** виплата 10⚛ при перших ключових досягненнях (перша колонія, перша легендарна знахідка, перше створіння) — замість лише XP.
6. **Додати ≥5 нових common pre-gen зображень** (нижче), щоб розширити безкоштовний пул без зміни монетизації.

## Поточний стан туторіалу (детальний огляд)

### Що вже є:
- **`CinematicIntro`** — стартовий інтро для нових гравців
- **`TutorialOverlay`** — 24 кроки (awakening → encyclopedia-explain)
- **`BuildingQuest`** — 5 кроків колонізації (post-evacuation)
- **`ArenaTutorial`** — специфічний туторіал арени
- **`UnlockPopupKind`**: 3 типи → `combat` (L10), `terraform` (L48), `buildings` (перша будівля)

### Критичні пробіли в існуючому туторіалі:
1. **24 кроки охоплюють L1 онбординг** — але після completion немає жодного навчального тригера до L10
2. **Carriers/транспорти** — не згадуються взагалі (ні в 24 кроках, ні в BuildingQuest)
3. **Operations Hub** (Beacon, Precursor, Seasons, Directives) — жоден пункт не має пояснення
4. **Tech Tree** — в туторіалі немає кроку "ось дерево технологій, ось що купити першим"
5. **Кварки** — прийнято за само-собою-зрозуміле, але нові гравці не знають джерел
6. **Galaxy map navigation** (кроки 14–18) — є, але `ast-probe` L16 unlock без пояснення

## Нотатки по конкретних фічах

### Carrier / Transport loop — детальна проблема

Нова гравиця:
1. Намагається запустити `deep_atmosphere_probe`
2. Отримує помилку `carrier_required` без пояснення
3. Не знає: (a) що carrier = окремий ресурс-юніт, (b) де його виготовити, (c) що він повертається сам після місії, (d) скільки часу виготовлення (35 хв для `atmo_probe_carrier`!)
4. **Виготовлення 35 хвилин** — це reeel pain без попередження

**Рекомендація:** quest + inline hint + показати таймер виготовлення в UI місії заздалегідь.

### Beacon (Мегаструктура) — детальна проблема

- Вимоги: 120k мінералів + 60k летючих + 32k ізотопів + 40k води = 252k одиниць
- Денний ліміт внеску: 500 одиниць
- При таких параметрах навіть при повній участі кластера з 50 гравців — сотні днів!
- Гравець бачить "BEACON" у Operations Hub, але не знає: (a) що це командна механіка, (b) яка його особиста вигода, (c) скільки йому ще ходити
- Відсутній прогрес-бар із поточним % заповнення — UI є, але пояснення — ні

### Precursor Cards — детальна проблема

- 14 карт, рідкість від common до legendary
- Drop rate: 15% (20% для deep_atmosphere_probe)
- Нові гравці абсолютно не знають про цю систему
- PrecursorGallery знаходиться у "Cosmic Archive → Collections" — неочевидний шлях
- Анімація фліпу (4.5s) є — але без контексту "що я щойно знайшов"

### Observation Seasons — детальна проблема

- 4 ротаційних сезони: pulsars, comets, nebulae, binaries
- Кожен сезон: 7 тижнів, 5 аномалій для complete, нагорода 150⚛
- Фінальні 72 год: 2× вага сезонних об'єктів
- Pity: після 6 скань без сезонного влучення → 4× вага
- SeasonHallGallery є, але жоден елемент UI не говорить "ти вже в сезоні, у тебе X/5"

### Система квестів (Академія) — що є vs що потрібно

Академія генерує квести через Gemini — контент хороший, але:
- Квести про реальні ігрові механіки (carrier, beacon, precursor) — відсутні
- Квести є окремо від основного онбордингу (не пов'язані з першим туторіалом)
- Гравець після туторіалу потрапляє у "void" без чіткого "що робити далі"

## Cosmic Events — Додаткові спостереження

### Чому "одна подія" — помилкове відчуття

Коли розробник каже "може тільки одна подія" — він, ймовірно, має на увазі:
- **Comet Herald** (`comet-herald`, `eventOnly`) — це єдина **time-limited live event** механіка (`comet-event.ts`)
- Основні 116 rollable об'єктів — це НЕ "події" у сенсі live event, це discovery-об'єкти
- Сезонні 20 аномалій — це **Observation Season** механіка, не окремі "events"

**Висновок:** механіка **live events** (типу Comet Herald) дійсно **одна**. Рекомендується додати:
- **Meteor Shower Event** (короткострокова, 48 год) — бонус до `small-bodies` відкриттів
- **Solar Storm Event** (тижнева) — збільшення Observatory discovery chance +20%
- **Intergalactic Signal Event** (місячна) — бонусні Precursor card drops
- **Cluster Nexus Event** (сезонна) — кооперативний challenge для кластеру → extra Beacon progress
